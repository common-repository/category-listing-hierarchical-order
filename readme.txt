Category Listing Hierarchical Order


== Description ==

This plugin use to Display Listing Post's Category in Hierarchical Tree Using Template
you want create new page in admin panel & select "List All Categories in Hierarchical Tree"
Template & Publish it. Now, You can view page for display all category listing in hierarchical.


Primary Features
*  Category Listing in Hierarchical Tree
*  A front-end Category Listing
*  This plugin is free


== Installation ==

Go to Plugins > Add New.
Search for "Category Listing Hierarchical Order".
Click "Install Now".

OR

Download the zip file.
Upload the zip file via Plugins > Add New > Upload.
Activate the plugin. Look for "Category Listing Hierarchical Order" in the admin menu.
